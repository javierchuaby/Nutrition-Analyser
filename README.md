# Starbucks Nutrition Analyser

An interactive web application for analyzing Starbucks menu nutrition data. Upload CSV files containing drinks and food items, apply filters, and get comprehensive nutrition analysis with AI-powered summaries and visualizations.

**Features:**

- Filter items by calories, sugar, protein, sodium, and more
- View interactive visualizations comparing drinks vs. food
- Get AI-powered nutrition summaries using Groq LLM
- Automatic data normalization and encoding detection
